# TASK-F020: 财务报表-收益表

## 目标
实现「财务报表-收益表」页面，展示当年持有收益、处置收益、利息支出、净收益。

## 功能需求

### 核心指标
- **持有收益**: 资产持有期间的价值变化 + 现金流入（如租金）
- **处置收益**: 资产出售时的盈亏
- **利息支出**: 负债每年的利息费用
- **净收益**: 持有收益 + 处置收益 - 利息支出

### 数据来源
- 持有收益: 资产 initData.cumulativeHoldReturn（需按年度分解）
- 处置收益: 资产 initData.cumulativeDisposeReturn
- 利息支出: 负债 initData.cumulativePaidInterest（负数表示已付）

### 实现方式
- 在 reports 页面新增一个 tab 切换：「资产负债表」/ 「收益表」
- 或在同一页面分区展示两种报表
- 年度选择器同步

### 验收标准
- [ ] 持有收益 / 处置收益 / 利息支出 / 净收益 四大指标清晰展示
- [ ] 年度切换时数据联动
- [ ] 无数据时显示 ¥0 万
