# TASK-F019: 财务报表-资产负债表

## 目标
实现「财务报表-资产负债表」页面，按年度展开资产合计、负债合计、净资产，自动从台账汇总。

## 交付物
1. `TASK-F019.md`（本文档）
2. `scripts/app.js` 修改 - 重写 `renderReports()` 方法
3. `styles/main.css` 修改 - 添加资产负债表样式
4. `feature_list.json` 更新 - F019 标记为 passes: true

## 功能需求

### 核心功能
- [ ] 年度选择器（2023-2030），高亮当前选中年份
- [ ] 资产分组小计：自用资产/投资资产/金融资产各自小计
- [ ] 负债分组小计：银行负债/非银行负债/私人负债各自小计
- [ ] 资产合计、负债合计、净资产（资产-负债）汇总行
- [ ] 自动从 `data.assets` 和 `data.liabilities` 台账汇总数据

### 数据逻辑
- 只显示 `buyYear <= 当前选中年份` 的资产/负债
- 资产按 `type` 分组：`selfUse`/`investment`/`financial`
- 负债按 `type` 分组：`bank`/`nonBank`/`private`
- 净资产 = 资产合计 - 负债合计

### UI 要求
- 卡片式布局，分为「资产」「负债」「净资产」三大板块
- 每块数据清晰，使用万元为单位，保留1位小数
- 资产合计/负债合计/净资产用醒目样式区分
- 可折叠/展开各分组（可选）
- 底部显示「数据自动汇总自资产台账」提示

## 实现计划
1. 重写 `renderReports()` 方法
2. 添加辅助方法 `getBalanceSheetData(year)` 返回结构化数据
3. 添加 CSS 样式
4. 更新 `feature_list.json`

## 验收标准
- [ ] 切换年份时，资产合计、负债合计、净资产正确更新
- [ ] 资产分组显示与小计正确
- [ ] 负债分组显示与小计正确
- [ ] 净资产 = 资产合计 - 负债合计
- [ ] 无资产/负债时显示 ¥0 万，而非 NaN 或空白
